## Descripcion
El programa detecta funciones detro del archivo programa.txt y las clasifica segun su estilo:
- Snake case
- Camel case
- Pascal case
- Desconocido
Durante la ejecucion del codigo utilice dos tipos de programa.txt, el primero fue el ejemplo del enunciado de la tarea, dando como resultado lo siguiente:
- clasificaba cada funcion dentro de su respectivo estilo, contando numero de funciones, variables declaradas, diferencias de estilo y errores de sintaxis.
El segundo programa.txt contenia funciones mas desordenadas (de ejemplo podria ser una funcion con un espacio antes de ()) pero clasificaba de igual forma las funciones segun el estilo en sus respectivos archivos.
NOTA: en ambos casos el archivo crea para el estilo Desconocido me lo entregaba vacio.
